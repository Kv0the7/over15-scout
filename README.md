# Over 1.5 Scout

Dashboard para detectar partidos de fútbol europeo con alta probabilidad de Over 1.5 goles.

## 🚀 Acceso directo

👉 **[Abrir la app](https://TU_USUARIO.github.io/over15-scout/)**

## Ligas incluidas
Premier League · La Liga · Serie A · Bundesliga · Ligue 1 · Eredivisie · Scottish Prem. · Championship · Liga Hypermotion

## Tecnología
- API-Football para fixtures y estadísticas
- The Odds API para cuotas reales de Bet365 y Winamax
- Algoritmo propio con peso aumentado en últimos 5 partidos
